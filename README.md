# duancodeweb_gym

Website phòng tập gym với thông tin về các khóa học và dịch vụ.

## Nội dung

- **Trang chủ** (`index.html`): Giới thiệu về phòng tập và các dịch vụ chính
- **Trang Nghiệp Dư** (`nghiep-du.html`): Thông tin chi tiết về chương trình đào tạo dành cho người mới bắt đầu
- **Styles** (`styles.css`): File CSS cho toàn bộ website

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt web
2. Điều hướng đến trang "Nghiệp Dư" để xem thông tin về khóa học cho người mới

## Tính năng

- Thiết kế responsive, phù hợp với mọi thiết bị
- Giao diện hiện đại với màu sắc chuyên nghiệp
- Thông tin chi tiết về chương trình nghiệp dư (amateur) cho người mới bắt đầu

## Công nghệ

- HTML5
- CSS3
- Thiết kế responsive
